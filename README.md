# Sistema Web para Controle de Egressos Aprovados na OAB

Sistema desenvolvido utilizando Bootstrap (AdminLTE) no frontend e PHP puro no backend.

A aplicação permite gerenciar alunos que concluíram a Faculdade de Direito e foram aprovados na OAB, oferecendo funcionalidades como:

- Importação de dados
- Acompanhamento de documentos
- Gestão de informações dos egressos
- Comunicação entre egressos

Projeto acadêmico com foco em organização, controle e gerenciamento de dados.
